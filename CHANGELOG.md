## Breaking Changes Policy

We follow semantic versioning strictly:
- **Major versions** may include breaking changes
- **Minor versions** are backward compatible
- **Patch versions** only include bug fixes and security patches

### Breaking Change Notice Period
- **90 days** advance notice for major breaking changes
- **30 days** notice for minor breaking changes
- **Immediate** for critical security fixes
