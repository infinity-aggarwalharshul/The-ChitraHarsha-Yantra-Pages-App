## ğŸš€ Quick Start

### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)
- Internet connection for CDN resources
- Optional: Firebase project for authentication

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/chitraharsha/yantra.git
   cd yantra
   ```

2. **Open the application**
   ```bash
   # Simply open index.html in your browser
   open index.html
   # Or use a local server
   python -m http.server 8000
   ```

3. **Configure Firebase (Optional)**
   ```javascript
   // Update firebaseConfig in index.html
   const firebaseConfig = {
     apiKey: "your-api-key",
     authDomain: "your-domain.firebaseapp.com",
     projectId: "your-project-id",
     // ... other config
   };
   ```

### Usage

1. **Access the Platform**: Open `index.html` in your browser
2. **Initialize System**: Click "INITIALIZE" to access the dashboard
3. **Innovation Search**: Use the "INNOVATION HUB" for patent analysis
4. **Cloud Management**: Monitor deployments and resources
5. **Intelligence Hub**: Access research articles and insights
